## Welcome to site about Vasya Pupkin

### How use this site (Flask APP)?

#### Create and activate virtual environment

```bash
virtualenv venv
source venv/bin/activate
```

#### Install library

```bash
pip install Flask
```

#### Create context

```bash
export FLASK_APP=app.py
export FLASK_ENV=development
```

#### Run dev-server of Flask

```bash
flask run
```
